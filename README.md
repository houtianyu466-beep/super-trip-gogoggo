# 项目名称
AI 周末旅行规划助手

## 📖 项目介绍
详细说明：
1. 平日放假出行旅游最大的痛点是什么？ 除了钱包和规划，还有当日气温天气吧。我将ds接入，由你的ai私人助手为你解决一切烦恼！

## ✨ 核心功能
- 功能1：旅游路线规划
- 功能2：当日气温，天气预报
- 功能3：衣食住行，一步推荐到位

## 🛠️ 技术栈
前端：
Vue 3 + Element Plus
后端：
Java 17
Spring Boot 3
LangChain4j
MySQL
Redis
MyBatis-Plus
数据库：MySQL / Redis / SQLite
