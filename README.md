# 项目名称
差旅行程规划审批agent

## 📖 项目介绍
详细说明：
1. 可用于公司出差员工日程规划以及多方案选择
## ✨ 核心功能
- 功能1：旅游路线规划
- 功能2：当日气温，天气预报
- 功能3：衣食住行，一步推荐到位

## 🛠️ 技术栈
前端：
Vue 3 + Element Plus
后端：
Java 17
Spring Boot 3
LangChain4j
MySQL
Redis
MyBatis-Plus
数据库：MySQL / Redis / SQLite


## 🚀 快速开始
### 环境要求
- JDK 17 
###部署方面
后端：云服务器 Docker 部署
前端：Nginx 部署
数据库：MySQL
